io.finch.Endpoint

StringOptionEndpointOps

Related Doc: package Endpoint

implicit final class StringOptionEndpointOps extends AnyVal

Implicit conversion that allows to call as[A] on any Endpoint[Option[String]] to perform a type conversion based on an implicit DecodeRequest[A] which must be in scope.

The resulting endpoint will fail when the result is non-empty and type conversion fails. It will succeed if the result is empty or type conversion succeeds.
